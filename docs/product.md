# MMGA 产品北极星

## 真正的意义

让学生**不在数学术语上犯困**，先听懂**到底在讲什么关系**，再碰课本说法。

不是刷题 App，不是搜题工具，不是教材百科。

## 成功标准

学完一小节，学生至少能做到：

1. **词**：用人话说明白关键的词（不背定义）
2. **事**：说得出这节在处理哪种关系或麻烦
3. **由**：说得出没有它会卡在哪里

## 第一次：先摸底

学生第一次使用时，默认先做一次**水平摸底**（不是排名考试）：

- 测的是「词和关系懂不懂」，不是题海分数
- 结果给出大约水平、建议起点课、薄弱点
- 之后 AI / 本地讲解都带上摸底结果；未摸底则按零基础术语处理

实现：`diagnostic.py` + `GET /diagnostic` + `POST /diagnostic/submit`；Android 首次进入强制摸底。

## 讲解硬顺序（代码必须遵守）

1. 先拆词（人话解释术语）
2. 再讲「到底在讲什么」
3. 为什么会出现
4. 例子
5. 最后才给课本正式说法

**永远不要默认学生已经知道你用的数学术语是什么意思。**  
**不要用一个未解释的新术语解释另一个新术语。**

## 偏航自检

新增功能前问：

- 它是否帮助学生先过词、再懂事？
- 它是否默认学生已经懂术语？（若是，砍掉或改）

若答案对不上北极星，即使竞品有，也不要优先做。
